# 量化投资选股项目学习计划

## 项目概述

本项目是一个基于机器学习的量化投资选股系统，包含数据获取与处理、机器学习模型训练、策略回测评估、Web界面展示等完整功能模块。系统支持随机森林、XGBoost、LightGBM等多种机器学习算法，实现了完整的技术指标计算、特征工程和策略回测功能。

**时间安排**：即日起至四月底（约8周）

## 小组间协作机制

### 协作节点安排

```
第3周末 ─────► 知识分享会（数据格式对接）
    │
第5周末 ─────► 中期进度汇报（技术文档交流）
    │
第7周末 ─────► 材料整合会（答辩准备）
    │
第8周末 ─────► 结项答辩
```

### 沟通方式

| 沟通类型 | 频率 | 形式 |
|---------|------|------|
| 组内讨论 | 每周2次 | 线上/线下会议 |
| 组间协调 | 每2周1次 | 联合会议 |
| 进度汇报 | 每周1次 | 文档提交 |
| 问题解决 | 随时 | 群内讨论 |

### 信息传递机制

| 信息类型 | 传递方式 | 负责人 |
|---------|---------|--------|
| 学习进度 | 周报文档 | 各组组长 |
| 技术问题 | 群内讨论 | 全体成员 |
| 接口规范 | 文档共享 | 相关组对接人 |
| 答辩材料 | 统一模板 | 项目整合组 |

## 学习建议

1. **循序渐进**：按照阶段安排逐步学习，不要跳跃
2. **动手实践**：边学边运行代码，加深理解
3. **做好笔记**：记录学习过程中的疑问和心得
4. **积极交流**：遇到问题及时在群内讨论
5. **定期总结**：每周末总结本周学习内容
