1.2 三大创新点

编号创新点创新类型核心贡献
①收益-夏普感知排序损失函数修改创新将金融收益目标直接嵌入排序损失函数，使排序优化与投资目标对齐

1.3 创新点之间的逻辑关系

创新点①（更好的损失函数）→ 训练出更优的排序模型

创新点②（多模型融合 + AI解读）→ 融合多个模型的排序结果，用LLM解释推荐逻辑

创新点③（系统对比评估）→ 用严格实验证明上述方法的有效性

故事线：更好的优化目标 → 更稳定的融合策略 → 更透明的决策解释 → 严格的实验验证

3.3 实验1：收益-夏普感知损失函数对比

目的：验证改进的损失函数是否比标准NDCG损失函数产生更好的投资绩效。

实验组设计

组别损失函数说明
Baseline-1标准 LambdaRank (NDCG)LightGBM默认lambdarank目标
Baseline-2标准 rank:ndcgXGBoost默认rank:ndcg目标
Exp-1a收益加权NDCG将增益函数从离散标签替换为连续收益率加权
Exp-1b收益加权NDCG + 夏普惩罚在Exp-1a基础上加入Top-K组合夏普比率惩罚项
Exp-1c收益加权NDCG + 夏普惩罚 + CVaR约束在Exp-1b基础上加入条件风险价值约束

损失函数数学定义

标准NDCG损失（Baseline）：

Lrank = −Σq (1/|Dq|) Σk=1K (2relk − 1) / log2(k+1)

其中 relk 为离散相关性标签（0-30整数）。

收益加权NDCG损失（Exp-1a）：

Lreturn = −Σq (1/Zq) Σk=1K g(rk) / log2(k+1)

其中增益函数 g(rk) = |rk|α · sign(rk) + C，rk 为第k只股票的未来一周收益率，α 为收益敏感度参数（默认1.0），C 为常数偏移确保增益非负。

夏普惩罚项（Exp-1b）：

Lsharpe = Lreturn + λ · max(0, −ŜTopK)

其中 ŜTopK 为Top-K组合的滚动夏普比率，λ 为惩罚系数（默认0.1）。

CVaR约束（Exp-1c）：

Lcvar = Lsharpe + γ · CVaRα(TopK)

其中 CVaRα 为Top-K组合在置信水平α（默认0.95）下的条件风险价值，γ 为风险厌恶系数（默认0.05）。

需要实现的代码

任务文件优先级
新增自定义目标函数 return_aware_objectivemodel_lightgbm.pyP0
新增自定义目标函数 return_aware_objectivemodel_xgboost.pyP0
新增超参数搜索空间（α, λ, γ）model_lightgbm.py / model_xgboost.pyP0

预期结果

指标BaselineExp-1aExp-1bExp-1c
NDCG@10基准持平或略降持平或略降略降
年化收益基准提升5-15%提升10-20%提升5-10%
夏普比率基准提升显著提升显著提升
最大回撤基准持平改善显著改善

关键论证逻辑：NDCG可能略降（因为优化目标不再是纯排序质量），但投资绩效（夏普、回撤）显著提升，证明损失函数改进使排序目标与投资目标对齐。

3.4 实验2：多模型排序融合对比

目的：验证RRF排序融合是否比单模型更稳定，比简单平均更有效。

实验组设计

组别方法说明
Baseline-LLightGBM LambdaRank 单模型使用Exp-1b的最优损失函数
Baseline-XXGBoost rank:ndcg 单模型使用Exp-1b的最优损失函数
Exp-2a简单平均融合两模型预测分数归一化后取平均
Exp-2bRRF融合Reciprocal Rank Fusion，k=60
Exp-2cStacking融合两模型预测分数为元特征，Ridge回归为元学习器
Exp-2d加权RRF融合根据验证集NDCG动态调整RRF权重

RRF融合公式

RRF(d) = Σm∈M 1 / (k + rankm(d))

其中 M 为模型集合，rankm(d) 为股票d在模型m中的排名，k 为平滑常数（默认60）。

3.5 实验3：LLM可解释推荐效果评估

目的：验证LLM结构化推荐是否比纯模型输出更有实用价值。

实验组设计

组别推荐方式说明
Baseline纯模型输出仅输出Top-K股票代码和得分
Exp-3a模型 + 特征重要性输出Top-K股票 + 各股票的关键驱动因子
Exp-3b模型 + 特征贡献分解输出Top-K股票 + 逐股票的SHAP近似贡献
Exp-3c模型 + 特征贡献 + LLM结构化推荐完整链路：排序→解释→AI建议

评估方法

  1. 定量评估：LLM推荐后的模拟执行收益 vs 纯模型推荐的模拟执行收益
  2. 定性评估：邀请3-5名有投资经验的评估者，对推荐的合理性打分（1-5分）
  3. 一致性评估：LLM推荐操作与模型排序信号的一致率

各部分写作要点

5.3 论文写作常见陷阱（新手必看！）

陷阱1：先写代码后想故事

❌ 错误：先把所有实验跑完，再想怎么写论文。

✅ 正确：先确定论文的核心论点（故事线），再设计实验来验证论点。实验是为论点服务的。

陷阱2：堆砌实验结果，缺乏分析

❌ 错误：放一堆表格，让读者自己看。

✅ 正确：每个实验结果后必须有2-3段分析，回答：结果说明了什么？为什么？与预期是否一致？

陷阱3：相关工作写成"流水账"

❌ 错误：A做了什么，B做了什么，C做了什么……没有逻辑串联。

✅ 正确：按问题组织相关工作，每个小节末尾指出已有工作的局限性，自然引出你的贡献。

陷阱4：过度承诺创新性

❌ 错误："本文首次提出……""开创性地……"

✅ 正确：用客观表述，如"本文将XX方法引入XX场景""在XX基础上改进了XX"。

陷阱5：忽略消融实验

❌ 错误：只展示最终方法的整体效果。

✅ 正确：逐步移除每个创新组件，展示移除后效果下降，证明每个组件都有贡献。

陷阱6：图表不清晰

❌ 错误：截图粘贴、分辨率低、字体太小。

✅ 正确：使用矢量图（PDF/EPS），字体≥8pt，色盲友好配色，每张图都能独立理解。

第七部分：关键公式汇总

7.1 收益-夏普感知排序损失函数

标准NDCG损失：

DCG@K = Σk=1K (2relk − 1) / log2(k+1)

NDCG@K = DCG@K / IDCG@K

Lrank = −NDCG@K

收益加权增益函数：

g(rk) = |rk|α · sign(rk) + C

其中 rk 为第k只股票的未来收益率，α ≥ 1 为收益敏感度参数，C 为常数偏移确保增益非负。

完整损失函数：

L = −Σq∈Q [ (1/Zq) Σk=1K g(rkq) / log2(k+1) ] + λ · max(0, −Ŝq) + γ · CVaRα(Πq)

其中：Q为截面集合，Zq为归一化常数，Ŝq为截面q的Top-K组合滚动夏普比率，Πq为截面q的Top-K组合收益率分布，λ为夏普惩罚系数，γ为CVaR风险厌恶系数。

7.2 RRF排序融合

RRF(d) = Σm∈M wm / (k + rankm(d))

加权RRF：wm = NDCGmval / Σm'∈M NDCGm'val

7.3 评估指标

夏普比率：Sharpe = E[R − Rf] / σ(R − Rf) × √252

最大回撤：MDD = maxt (Peakt − Valleyt) / Peakt

卡尔玛比率：Calmar = 年化收益率 / 最大回撤
