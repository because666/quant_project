## 4. 问题诊断

### 双模型年化收益均为负

诊断分析：

1. **市场环境**：B-EW（随机选股）年化收益为 -6.5%，说明测试期市场整体偏弱
2. **模型 vs 随机**：B-LGBM(-25.2%) vs B-EW(-6.5%)，模型表现差于随机 → 模型选股偏向高波动股
3. **NDCG水平**：A股周频排序学习NDCG@10在0.3-0.4属正常水平，0.75-0.90为学术文献中信息检索任务的标准，不适用于股票排序
4. **核心问题**：模型选出的Top N股票在下跌市场中跌幅更大（高波动暴露）

改进方向：
- 添加波动率因子或风险调整因子
- 使用收益-夏普感知损失函数（实验1）
- 多模型融合降低选股波动（实验2）
- 考虑加入择时/空仓机制
